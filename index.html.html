ЦИФРОВОЙ ВОРКБУК ПО МАТЕМАТИКЕ

СПРАВОЧНИК: ФОРМУЛЫ ПРИВЕДЕНИЯ

Правило двух шагов:
1. Если опорная точка находится на вертикальной оси (\(\frac{\pi}{2}\), \(\frac{3\pi}{2}\)), функция меняется на противоположную (sin ↔ cos, tg ↔ ctg). Если на горизонтальной (\(\pi\), \(2\pi\)), функция сохраняется.
2. Знак перед итоговой функцией определяется знаком исходной функции в соответствующей четверти тригонометрического круга.
1. Опорная точка \(\frac{\pi}{2}\) (функция меняется):
\(\sin\left(\frac{\pi}{2} - \alpha\right) = \cos\alpha\)
\(\sin\left(\frac{\pi}{2} + \alpha\right) = \cos\alpha\)
\(\cos\left(\frac{\pi}{2} - \alpha\right) = \sin\alpha\)
\(\cos\left(\frac{\pi}{2} + \alpha\right) = -\sin\alpha\)
\(\tan\left(\frac{\pi}{2} - \alpha\right) = \cot\alpha\)
\(\tan\left(\frac{\pi}{2} + \alpha\right) = -\cot\alpha\)
2. Опорная точка \(\pi\) (функция сохраняется):
\(\sin(\pi - \alpha) = \sin\alpha\)
\(\sin(\pi + \alpha) = -\sin\alpha\)
\(\cos(\pi - \alpha) = -\cos\alpha\)
\(\cos(\pi + \alpha) = -\cos\alpha\)

ЗАДАНИЕ №6

Условие: Найдите производную функции \(y = \cot\left(\frac{\pi}{2} - 2x\right)\) в точке \(x_0 = \frac{\pi}{3}\).

Твой текст разбора: Сначала упрощаю, \(\frac{1}{2}\pi\) число 1 умножаем на \(\pi\) и делим на 2. Получилось \(\cot\left(\frac{\pi}{2} - 2x\right)\). Точка \(\frac{\pi}{2}\) (90°) находится на вертикали тригонометрического круга, значит функция меняется.
1. Упрощение по формуле приведения:
\[y = \cot\left(\frac{\pi}{2} - 2x\right) = \tan(2x)\]
Если мы от 90 градусов отнимем любой острый угол альфы, мы окажемся в первой четверти. Там тангенс положителен, поэтому знак сохраняется.
2. Взятие производной сложной функции:
\[y' = \frac{1}{\cos^2(2x)} \cdot 2 = \frac{2}{\cos^2(2x)}\]
3. Подставление точки:
\[y'\left(\frac{\pi}{3}\right) = \frac{2}{\cos^2\left(\frac{2\pi}{3}\right)}\]
Находим косинус: \(\cos\left(\frac{2\pi}{3}\right) = -\frac{1}{2}\). Возводим в квадрат: \(\left(-\frac{1}{2}\right)^2 = \frac{1}{4}\).
\[y'\left(\frac{\pi}{3}\right) = \frac{2}{1/4} = 8\]

Ответ: b) 8

ЗАДАНИЕ №7

Условие: Найдите тангенс угла наклона касательной к функции \(y = -3x^3 + x^2\) в точке \(x_0 = 1\).

Геометрический смысл: Тангенс угла наклона равен значению производной в данной точке. Шаблон формулы: \((x^n)' = n \cdot x^{n-1}\)
Решение:
\[y' = (-3x^3 + x^2)' = -9x^2 + 2x\]
Подставляем точку \(x_0 = 1\):
\[y'(1) = -9 \cdot 1^2 + 2 \cdot 1 = -7\]

Ответ: b) -7

ЗАДАНИЕ №8

Условие: Решите неравенство \(\frac{x + 5}{2x + 1} < 1\).

Важное правило: На знаменатель умножать нельзя, переносим 1 влево к нулю и приводим к общему знаменателю.
Пошаговое приведение:
\[\frac{x + 5}{2x + 1} - 1 < 0\] \[\frac{(x + 5) - (2x + 1)}{2x + 1} < 0\] \[\frac{-x + 4}{2x + 1} < 0\]
Критические точки: числитель равен нулю при \(x = 4\), знаменатель равен нулю при \(x = -\frac{1}{2}\). Используя метод интервалов, получаем верный диапазон.

Ответ: a) \((-\infty; -0.5) \cup (4; +\infty)\)

ЗАДАНИЕ №9

Условие: Найдите сумму наибольшего и наименьшего значений функции \(f(x) = 6x^4 - 3x^2\) на отрезке \([0; 1]\).

1. Поиск производной:
\[f'(x) = 24x^3 - 6x\]
2. Приравниваем к нулю и находим точки:
\[6x(4x^2 - 1) = 0\]
Корни: \(x = 0\), \(x = 0.5\), \(x = -0.5\) (не входит в отрезок).
3. Вычисляем значения на границах и в экстремумах:
\[f(0) = 0\]
\[f\left(\frac{1}{2}\right) = 6\left(\frac{1}{16}\right) - 3\left(\frac{1}{4}\right) = \frac{3}{8} - \frac{6}{8} = -\frac{3}{8}\] (наименьшее)
\[f(1) = 6(1) - 3(1) = 3\] (наибольшее)
4. Сумма экстремумов:
\[3 + \left(-\frac{3}{8}\right) = \frac{21}{8}\]

Ответ: b) 21/8

ЗАДАНИЕ №23

Условие: Радиус кругового сектора равен 8 см, а его угол — 45°. Сектор свернут в коническую поверхность. Найдите площадь основания конуса.

Длина дуги исходного сектора равна длине окружности получившегося основания конуса:
\[2\pi r = \frac{2\pi \cdot 8 \cdot 45^\circ}{360^\circ} = 2\pi \implies r = 1\text{ см}\]
Площадь круглого основания: \(S = \pi r^2 = \pi \cdot 1^2 = \pi\).

Ответ: c) \(\pi\text{ см}^2\)
